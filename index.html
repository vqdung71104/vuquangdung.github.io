Student Management System

Full-Stack Application

A comprehensive academic management platform where students can upload their academic progress and receive intelligent course recommendations with sugestive subjects and optimized classes. Featuring with a Hybrid Chatbot with Intent Classification using TF-IDF, Word2Vec, and Cosine Similarity for natural language queries. Implements NL2SQL service with 45 SQL templates for database interaction. Includes intelligent academic counseling Rule Engine that recommends courses based on curriculum roadmap, failed courses, and academic warning levels. Features schedule optimization using itertools.product algorithm to generate conflict-free timetables with Top 3 recommendations based on personal preferences. Built with three-tier architecture using FastAPI (async backend), React 19 with Ant Design 5 and Tailwind CSS (frontend), Chart.js for GPA visualization, and MySQL database (14 tables via SQLAlchemy ORM). Secured with JWT authentication, Bcrypt password hashing, and email OTP verification. Supports bulk data upload via Excel files for both administrators and students.
